# PokéAPI en Vue

Crea una aplicación Vue 3 que consuma el API REST de [PokéAPI](https://pokeapi.co/docs/v2).

## Tareas a realizar

- Consume por lo menos dos endpoints diferentes (por ejemplo [Pokemon](https://pokeapi.co/docs/v2#pokemon)
  o [Abilities](https://pokeapi.co/docs/v2#abilities)).
- Utiliza [Axios](https://axios-http.com) para hacer las peticiones.
- Divide el interfaz de usuario de la aplicación en componentes independientes (por ejemplo, cabecera, barra de navegación y bloques de contenido).
- Utiliza el router Vue para hacer la navegación interna de la aplicación.
- Usa Bootstrap 5 como framework para desarrollar el GUI.
